1. Base de datos de clientes

Esta es la base de datos principal, la misma refleja todos los clientes tanto activos como potenciales del sistema. A traves de esta interfaz se puede aceeder a un perfil del cliente donde se guarda información y documentos del mismo

En en panel principal se pueden realizar las siguientes acciones:

  • Crear nuevos clientes, cuyo estado inicial siempre sera "Potencial"
  • Ingresar al perfil de cada usuario
  • Borrar usuarios, pero se debe tener presente que esta acción eliminara toda la información del mismo, como pagos o prestamos asociados
  • Activar la función recalculo, que permite recalcular la situaciones de todos los clientes en caso de encontrar un error

Por otro lado, todos los clientes tienen un perfil en el que se pueden realizar las siguientes acciones:

  • Editar la información personal del cliente
  • Ver información resumida de los creditos del mismo
  • Crear notificaciones que se crearan automaticamente en un calendario Google ya predefinido
  • Ver resultados del programa MAPAFER
  • Cargar documentos asociados

2. Situaciones posibles de los clientes

Las situaciones estan vinculadas al comportamiento de un cliente con los prestamos, y las mismas se calculan automaticamente de la siguiente manera

  • Situación 1: Se mantiene cuando no tiene deuda mayor a 30 dias
  • Situación 2: Se mantiene cuando no tiene deuda mayor a 90 dias
  • Situación 3: Se mantiene cuando no tiene deuda mayor a 180 dias
  • Situación 4: Se mantiene cuando no tiene deuda mayor a 365 dias
  • Situación 5: La deuda supera los 365 dias
  • Situación 6: Sin definir
